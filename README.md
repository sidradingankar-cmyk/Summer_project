# AI Campus Navigation System

This repository contains the **AI Campus Navigation System** project, a web-based application designed to help users navigate a university campus using AI-powered routing and points of interest.

## Features
- Interactive map with searchable buildings and locations
- Real‑time pathfinding powered by AI algorithms
- Responsive design for desktop and mobile devices
- Dark mode and modern UI using glassmorphism aesthetics

## Getting Started
1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ai-campus-navigation-system.git
   cd ai-campus-navigation-system
   ```
2. **Install dependencies** (assuming a Node.js environment):
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.
